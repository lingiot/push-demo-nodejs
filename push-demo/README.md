# 灵联云推送配置 Node.js Demo

第一步：下载项目源码，或使用 Git clone：

```
git clone https://github.com/lingiot/push-demo-nodejs.git
```

第二步：进入项目根目录，安装依赖：

```
cd push-demo-nodejs
npm i
```

第三步：启动项目

```
npm start
```

第四步：在「系统信息」-> 「开发配置」菜单下配置推送 URL，地址为

```
{YOUR_SERVER_HOST}/pushCallback
```

比如：

```
https://push-demo.linxot.com/pushCallback
```

好了。
